# 005-JS简介 小结

## JavaScript是什么？

JavaScript是一门编程语言，可以实现很多的网页交互效果

## 2.JavaScript 书写位置?

内部JavaScript

内部JavaScript - 写到 </body> 标签上方

外部javaScript - 但是 <script> 标签不要写内容，否则会被忽略

## 3.JavaScript 的注释?

单行注释 //

多行注释 /**/

## 4.JavaScript 的结束符?

分号; 可以加也可以不加，可以按照团队约定

## 5.JavaScript 输入输出语句?

### 输入: 

~~~javascript
prompt()
~~~

### 输出：

~~~javascript
alert()
document.write()
console.log()
~~~
